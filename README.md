## 🤺Fenjoy
# フェンシングプレイヤーみんなのためのアプリケーション
## 🗣概要
簡単にフェンシング用の試合表を作成することができます
## 🔗URL
https://fenjoy.herokuapp.com/

## 👩‍💻利用方法
☆トップページから表作成ボタンを押しフォーム画面へ移動
☆人数と選手の名前(人数分)を入力し、表の表示画面へ移動
☆表の作成画面下のボタンから表を保存

## 課題解決
c
| フェンシングの試合の記録用紙が見づらい(試合順番)     | 人数と名前を入力するだけで対戦の順番が可視化できる! |
| 記録用紙が管理しづらい                           |　アプリケーションにすることによって、出先でも使用可能かつ保存できることによって、端末での管理ができる｜

## 要件
|  機能              |  目的                                                              |
| ----------------- | ------------------------------------------------------------------ |
| 試合の順番表示機能  | 試合の記録用紙の試合の順番が数字で書かれており、見にくいという問題を解決するため  |
| 記録用紙作成機能    | 記録用紙と筆記用具の用意の手間が省けることや端末で管理するため                 |
| 記録用紙保存機能    | 作成した記録用紙を保存し、スマホで管理できるようにするため                     |

## 🔒実装完了した機能
  試合順番表示機能
  試合表記述機能
